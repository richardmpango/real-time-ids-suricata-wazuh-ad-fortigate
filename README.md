# real-time-ids-suricata-wazuh-ad-fortigate
A real-time intrusion detection and security monitoring lab using Suricata, Wazuh, Active Directory, FortiGate firewall, and VMware.
## Academic Context

This project is related to the **Summer 2026 INCS 615 Advanced Network and Internet Security Term Project**. The course project allows students to choose a topic related to network security, internet security, or any subject connected to the topics covered in class.

This repository follows **Option 2: Project Demo**, where the required submission includes a GitHub link or project code and a 2-page IEEE-format implementation report. The project demonstrates a practical real-time intrusion detection and security monitoring lab using Suricata, Wazuh, Active Directory, FortiGate firewall, and VMware virtual machines.

The selected topic is connected to the course topic **Real-Time Intrusion Detection System**. It also relates to network security monitoring, intrusion detection, firewall logging, authentication monitoring, and security event correlation. The implementation includes a lab architecture, safe traffic generation, Suricata IDS alerts, Wazuh monitoring, Active Directory security events, FortiGate firewall logs, and analysis of the expected results.

The goal of this project is to show how open-source and enterprise-style security tools can be combined to detect suspicious activity in a small network environment. The project includes implementation notes, configuration examples, test scenarios, screenshots, and results that support the final 2-page IEEE-format report.
